Youstagram

Technologies: Express.js, Flexbox, Instagram API, JS, EJS, Heroku

Have you ever wanted to change the interface of a popular app?

Now you can! Youstagram pulls your data from Instagrams API and puts it in a new layout. This project got me writing some backend javascript, deploying, flexboxing and playing with some new HTML features like the data- attribute
