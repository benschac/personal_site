NHL Player Search

Technologies: React + D3.js

Search by team, age or name and you'll see a count of nhl players matching your query

With it's speedy diffing engine, React and D3.js are a solid team for realtime data visualizations. This application uses React's life-cycle components to check the DOM for any updates. Some other fun solutions that came out of this project was filtering and counting the raw data from the API using functional javascript.
